### What Makes Us Different

Most code review tools use **one generic AI model** that gives generic feedback. Hostile Review deploys **108 specialized agents** — each one an expert in a different domain:

- **Security** — SQL injection, XSS, hardcoded secrets, authentication flaws, SSRF, path traversal
- **Performance** — N+1 queries, memory leaks, unnecessary allocations, blocking I/O
- **Architecture** — coupling, separation of concerns, SOLID violations, dead code
- **Reliability** — race conditions, error handling, null safety, edge cases
- **Compliance** — OWASP Top 10, PCI-DSS patterns, GDPR data handling
- **And 100+ more** — i18n, accessibility, logging, testing gaps, dependency risks

Problems that a single model misses, specialized agents catch.

### How It Works

1. **Install** the [Hostile Review GitHub App](https://github.com/apps/hostilereview) on your repos
2. **Open a PR** — 108 agents automatically review the diff
3. **Get findings** posted as inline comments directly on your code
4. **Fix issues** with one-click copy to your AI coding assistant

Every push triggers a fresh review. No configuration needed — it works on any language, any framework, any repo size.
